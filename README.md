 # Arz Media Swift

ArzMedia allows you to see currency rates from PHP and MYSQL services  and cache the results.


### Features

* Installs on IOS devices.
* see currency rate . 
* user can check Currency exchange on Google Map
* user can add Currency exchange on Map
